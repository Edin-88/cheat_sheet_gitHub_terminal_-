Git Fehlerbehebung

How to fix: Error Permission Denied (publickey) |Fatal: Could Not Read From Remote Repository

Gib folgendes ins Terminal ein: ssh-keygen -t rsa

Gib den Dateinnamen ein z.B. "12345"cund drücke auf y

Gib dann folgenden Befehl ins Terminal ein pbcopy < ~/.ssh/id_rsa.pub

Gehe dann ins Settings bei GitHub. Wähle SSH and GPPG keys.
Klicke auf New SSH key und drücke command v um den key einzufügen
